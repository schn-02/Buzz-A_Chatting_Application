# Buzz - A Chatting Application

Buzz is a real-time Android chat application built using **Java, XML, Firebase, and MVVM architecture**.  
It allows users to sign up, log in, view other users, and chat in real time with a clean and responsive UI.

## 🚀 Features

- User registration and login
- Real-time one-to-one chatting
- Firebase Authentication
- Firebase Firestore/Realtime Database integration
- Clean Android XML UI
- MVVM-based project structure
- Logout functionality

---

## 🛠️ Tech Stack

- **Language:** Java
- **UI:** XML
- **Architecture:** MVVM
- **Backend/Database:** Firebase
- **Authentication:** Firebase Auth
- **IDE:** Android Studio
